# 📸 이미지 용량 압축기
* 대용량 이미지 파일을 효율적으로 처리하기 위해 개발된 Python 기반 데스크톱 애플리케이션
* 사용자가 지정한 특정 이미지 파일을 실시간으로 감시하여 압축하거나, 여러 파일을 드래그 앤 드롭하여 한 번에 처리
* 저장 폴더 설정 없이, 결과물을 항상 원본 파일과 동일한 위치에 생성

## ✨ 주요 기능
* **실시간 자동 감시)**: 설정된 폴더 내에서 특정 파일(예: `screenshot.jpg`)이 생성되거나 수정되면 즉시 감지하여 처리
* **스마트 압축 로직 (10MB 기준)**:
* **10MB 초과 시**: Quality 85% 압축을 진행하여 용량을 대폭 줄임
* **10MB 이하 시**: 원본의 화질을 보존하기 위해 파일명 뒤에 `_original`을 붙여 복사본을 생성

* **드래그 앤 드롭 지원**: `tkinterdnd2`를 활용하여 파일을 프로그램 창 위로 끌어다 놓는 것만으로 간편하게 목록에 추가할 수 있음
* **설정 자동 저장**: 마지막으로 설정한 감시 폴더와 타겟 파일명은 `compressor_config.json`에 저장되어 프로그램 재시작 시 자동으로 불러올 수 있음
* **크로스 플랫폼 지원**: Windows, macOS, Linux 환경에서 모두 동작하며 각 OS에 맞는 폴더 열기 기능을 제공


## 🚀 사용 방법

### 1. 수동 처리 모드
1. `파일 선택 (수동)` 버튼을 누르거나 이미지 파일(.png, .jpg, .jpeg)을 프로그램 창으로 드래그
2. 목록에 추가된 파일을 확인한 후 `선택 파일 처리 시작` 버튼을 클릭
3. 원본 폴더에 `_compressed` 또는 `_original`이 붙은 결과 파일이 생성됨

### 2. 자동 감시 모드
1. `폴더 찾기` 버튼을 눌러 감시할 폴더를 선택
2. 감지할 파일 이름을 확장자를 포함하여 입력
3. `감시 시작` 버튼을 클릭하면 프로그램이 백그라운드에서 해당 파일의 생성을 대기함

## 📂 파일 구조
* `image_compressor.py`: 메인 실행 소스 코드
* `compressor_config.json`: 사용자 설정 저장 파일 (자동 생성됨)
* `.gitignore`: `build/`, `dist/`, `config` 등 불필요한 파일의 버전 관리 제외 설정

## ⚠️ 참고 사항
* **RGBA 처리**: 투명 값이 있는 PNG 파일 압축 시, 자동으로 RGB 모드로 변환하여 JPEG로 저장
* **파일 잠금 방지**: 파일 생성 직후 처리 시 발생할 수 있는 오류를 방지하기 위해 0.5초의 지연 시간을 두고 작업을 수행
